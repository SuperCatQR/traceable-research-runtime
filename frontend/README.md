# 迹研前端 Demo

该工程只展示本地 typed fixtures，不请求 Rust Runtime，不包含 mock HTTP 服务，也不把组件状态当作未来 wire contract。

```powershell
npm install
npm run dev
```

质量检查：

```powershell
npm run test
npm run build
```

产品、交互、页面、未来接口和视觉决策见 [`../docs/dev/frontend-demo-product-draft.md`](../docs/dev/frontend-demo-product-draft.md)。
